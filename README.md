## 功能

* vue-cli4 构建
* axios 请求、响应拦截,封装
* weeppack配置 生产、测试、开发 三套环境
* 首屏加载等待动画
* 分离的路由和菜单设置
* 可折叠侧边栏
* 全屏
* 模拟数据
* 多标签页： 支持鼠标右键点击 刷新、关闭功能、可拖拽
* 模块化全局状态管理
* 数据持久化：实现用户登录、路由菜单
* 切换全局基础组件尺寸 
* 页面载入进度条
